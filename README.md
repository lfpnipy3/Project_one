### Описание на чём написан проект
- Разработка кроссплатформенного приложения (Android/iOS/Web) на Flutter.  
- Бэкенд реализован на C# (ASP.NET Core Web API), обеспечивая RESTful взаимодействие.  
- Данные хранятся в PostgreSQL, доступ к API задокументирован через Swagger.
