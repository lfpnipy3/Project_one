## **Описание на чём написан проект**
- Разработка кроссплатформенного приложения (Android/iOS/Web) на Flutter
- Бэкенд реализован на C# (ASP.NET Core Web API), обеспечивая RESTful взаимодействие
- Данные хранятся в PostgreSQL, доступ к API задокументирован через Swagger
- Шифрование Argon2  


## **Технические требования к ПК**
На данный момент, а именно 21.04.2026 технических характеристик к разработке не достаточно (нагрузка уходит на процессор, нехватка оперативки).  
- Процессор: Intel I5 760 (LGA 1156)
- Материнская плата: ASUS P7P55D-E PRO
- Оперативная память: 8 GB, DDR3-1332 / PC3-10700 DDR3 SDRAM UDIMM Kingston
- HDD: 500 GB, TOSHIBA DT01ACA050
- Видеокарта: ASUS ENGTX580
- Кулер: Thermaltake Frio (CLP0564)
- Корпус: Thermaltake
